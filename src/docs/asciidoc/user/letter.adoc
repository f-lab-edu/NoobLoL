== introduction

해당 API는 쪽지에 대한 기능을 작성한 문서입니다.

정리된 기능은 쪽지의 발송, 삭제, 단건 조회, 리스트(문서함)조회 API문서입니다.

추가적으로 쪽지의 모든 기능은 **로그인**이 되어 있어야 사용이 가능합니다
